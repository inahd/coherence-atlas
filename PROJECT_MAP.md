# Atlas Project Map

## Core directories

- `atlas/` — core Python modules and runtime pieces
- `scripts/` — operational tools and pipelines
- `tools/` — generators/helpers
- `data/` — nodes, relations, datasets
- `memory/` — graph snapshots, reports, visual artifacts
- `docs/` — architecture, canon, philosophy, onboarding
- `research/` — ingested/experimental knowledge materials
- `visual/` — visualization outputs and assets
- `cards/` — card generation experiments

## Important modules

- `atlas/ui/` — ncurses interface components
- `atlas/core/` — graph loading, saving, and core utilities
- `scripts/resolve_layers.py` — relation resolution to canon/inference/overlay
- `scripts/build_canonical_graph_ideal.py` — canonical graph materialization

## Current priorities

1. repository stability
2. graph relational density
