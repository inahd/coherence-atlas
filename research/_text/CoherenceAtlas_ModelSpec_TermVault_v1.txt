Nak■atra, Pak■a, Samaya, ■tu), producing transparent indices (Mobility/Heat/Cohesion), VPK & Gu■a
vectors, and an auditable rationale trail.

meanings + how each term is used in the model).
This is not presented as deterministic fate or prediction. It is a structured model for examining temporal
modulation of nature and behavior (prak■ti/gu■a/do■a rhythms) with devotional framing available as an
interpretive layer.

2. Core Derivations (Astronomy → Tithi/Pak■a/Nak■atra)
Let λ■ be Sun ecliptic longitude, λ■ be Moon ecliptic longitude (tropical or sidereal).
Phase angle: φ = (λ■ − λ■) mod 360.
Tithi(30): t30 = floor(φ / 12°) + 1.   (Ranges 1..30)
Pak■a: ■ukla if φ < 180°, else K■■■a.
For a 15-fold devotional/music mapping: t15 = ((t30 − 1) mod 15) + 1.
Nak■atra(27): n = floor( (λ■ mod 360) / 13°20′ ) + 1.
Where 13°20′ = 13.333333... degrees.
Sidereal mode: λ_sidereal = (λ_tropical − ayan■■■a) mod 360. In v1 we use a Lahiri-style ayan■■■a
approximation plus an optional fine-tune offset for matching a chosen pañc■■ga reference.

3. Indices → VPK (Mobility / Heat / Cohesion)
We define three continuous indices. These are designed to be tunable, transparent, and auditable.
MobilityIndex (V■ta-like): movement, variability, subtlety, dispersion.
HeatIndex (Pitta-like): sharpness, transformation, combustion, intensity.
CohesionIndex (Kapha-like): stability, nourishment, cohesion, inertia (when excessive).
VPK vector is the normalized form of these indices:
V = MobilityIndex / (Mobility+Heat+Cohesion)
P = HeatIndex / (Mobility+Heat+Cohesion)
K = CohesionIndex / (Mobility+Heat+Cohesion)
Index decomposition:
Index = PhaseComponent(Tithi/Pak■a) + SamayaComponent(time-of-day) + RituComponent(season) + NakshatraWeight + (optional) LunarSpeedComponent
