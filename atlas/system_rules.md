# Atlas System Rules (for Coding Agents)

1. **Graph first**
   - Treat graph artifacts as truth layer; UI and generated content are projections.

2. **No parallel truth stores**
   - Do not create duplicate node/relation sources when a canonical source exists.

3. **Small coherent patches**
   - One cycle = one coherent improvement with verification commands.

4. **Preserve layer distinction**
   - Keep canonical ontology, instance data, and activation/time layers separate.

5. **Show measurable movement**
   - Prefer changes that produce visible outcomes (counts, explorer behavior, generated outputs).
