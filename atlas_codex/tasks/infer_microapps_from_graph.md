ATLAS CODEX TASK

MODE: Shilpi

GOAL

Create an inference engine that scans the canonical Atlas graph
and proposes possible micro-applications based on relation patterns.

Atlas should not require manual invention of tools.
Tools should emerge from graph structure.

------------------------------------------------

INPUT

memory/graphs/canonical_graph.json

Graph contains:

nodes
edges

Each node may include:

id
label
type
domain
spectral_layer (S0-S6)

------------------------------------------------

FILES TO CREATE

scripts/infer_microapps.py

atlas_codex/outputs/microapp_proposals.json

------------------------------------------------

INFERENCE RULES

Detect patterns in graph:

1. SOUND PATTERNS

If nodes contain:

mantra
raga
tala
rhythm

Propose:

tala_chakra_player
mantra_resonance_engine
cycle_visualizer

------------------------------------------------

2. COSMOLOGY PATTERNS

If nodes contain:

graha
nakshatra
planet
cycle

Propose:

jyotish_cycle_engine
nakshatra_explorer
planetary_transit_visualizer

------------------------------------------------

3. GEOMETRY PATTERNS

If nodes contain:

yantra
mandala
symbol
geometry

Propose:

yantra_generator
mandala_explorer
symbol_projection_engine

------------------------------------------------

4. ECOLOGY PATTERNS

If nodes contain:

plant
pfaf
ecosystem
soil

Propose:

plant_guild_generator
ecosystem_simulator
permaculture_planner

------------------------------------------------

5. RELATIONAL GRAPH PATTERNS

If nodes contain dense relation clusters

Propose:

constellation_explorer
graph_cluster_viewer
relation_field_map

------------------------------------------------

OUTPUT FORMAT

atlas_codex/outputs/microapp_proposals.json

Example:

[
  {
    "name": "tala_chakra_player",
    "trigger_nodes": ["tala", "rhythm"],
    "spectral_layer": "S3",
    "description": "Circular rhythm visualization and playback"
  },
  {
    "name": "nakshatra_explorer",
    "trigger_nodes": ["nakshatra"],
    "spectral_layer": "S1",
    "description": "Explore nakshatra mythic and astronomical relations"
  }
]

------------------------------------------------

SUCCESS CONDITION

Running:

python scripts/infer_microapps.py

produces:

atlas_codex/outputs/microapp_proposals.json

with automatically inferred application ideas.

------------------------------------------------

CONSTRAINTS

Do not modify graph data.
Do not create new nodes.
Only generate proposals.

------------------------------------------------

OBJECTIVE

Atlas should gradually evolve a library of tools
derived from the knowledge graph itself.

Graph → instrument.
