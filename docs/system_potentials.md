# Atlas System Potentials

Atlas can generate structured outputs from the knowledge graph.

Possible outputs:

• Nakshatra cards
• Deity cards
• Graha cards
• Cosmology diagrams
• Research dossiers
• Infographics
• Teaching charts

Media generation pipeline:

graph entity
↓
structured attributes
↓
template
↓
media artifact

Example:

Entity: Rohini

Fields:
- type: nakshatra
- ruler: Moon
- deity: Brahma
- qualities: fertility, growth, beauty

These can render into:

• cards
• posters
• symbolic diagrams
